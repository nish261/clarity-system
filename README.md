# Clarity System

This repository contains a personal collection of Raycast scripts and an Obsidian vault for enhanced productivity and knowledge management.

## Contents

- **Raycast Scripts**: A set of scripts to streamline common tasks.
- **Obsidian Vault**: A pre-configured vault for note-taking and personal knowledge management.
- **Intent Enforcer**: A web-related application or browser extension.

---

## Raycast Scripts

### Location

The Raycast scripts are located in the `raycast script folder/godmode` directory.

### Setup

To use these scripts, you need to add the `godmode` directory to your Raycast script commands:

1.  Open Raycast and go to `Settings`.
2.  Navigate to the `Extensions` tab.
3.  Select `Script Commands`.
4.  Click `Add Directory` and choose the `raycast script folder/godmode` directory from this repository.

### Available Scripts

-   **`add-task.js`**: Adds a new task.
-   **`create-business.js`**: Creates a new business-related note.
-   **`create-journal.js`**: Creates a new journal entry.
-   **`create-note.js`**: Creates a new general note.
-   **`create-task.js`**: Creates a new task.

## Obsidian Vault

### Location

The Obsidian vault is located in the `your obsidian vault` directory.

### Setup

To use the vault, open the `your obsidian vault` directory in the Obsidian application:

1.  Open Obsidian.
2.  Click `Open another vault`.
3.  Select `Open folder as vault`.
4.  Choose the `your obsidian vault` directory from this repository.

### Vault Structure

The vault is organized with the following folders:

-   **`3. Tasks`**: For managing tasks.
-   **`5. References`**: For storing reference materials.
-   **`6. Tags`**: For organizing notes with tags.
-   **`7. Templates`**: For reusable note templates.
